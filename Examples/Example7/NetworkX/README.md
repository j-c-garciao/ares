# NetworkX Model in Python

This repository contains a Python implementation of a graph/network
model built using **NetworkX**.\
It includes scripts for creating the graph, computing flow and path
metrics, and visualizing the network.

## Requirements

### Python (3.8 or newer)

Download from: https://www.python.org/downloads/

### Python Packages

Install using:

``` bash
pip install -r requirements.txt
```

Typical packages:

    networkx
    matplotlib
    pandas

Optional packages:

    pyvis
    plotly

------------------------------------------------------------------------

## Setting Up NetworkX

Install NetworkX:

``` bash
pip install networkx
```

Verify installation:

``` python
import networkx as nx
print(nx.__version__)
```
## How to Run the Model

Run the main script:

``` bash
python network_model.py
```

## Model Output

Example:

``` bash
    Max flow from XX to YY: 2500
```
